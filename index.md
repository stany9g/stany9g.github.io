### Data types
* INT - integer - numbers
* VARCHAR - string - text
* DATETIME - date + time
* ...

### OPTIONS
* PRIMARY KEY
* FOREIGN KEY
* NOT NULL
* IDENTITY(1,1)

### Creating database

```SQL

CREATE DATABASE name;

```

### Creating table

```SQL
CREATE TABLE name (
  column_name TYPE(length) OPTIONS,
  ....

```

### Delete the table 

```SQL

DROP TABLE name;

```
